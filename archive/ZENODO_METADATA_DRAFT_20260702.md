## Description

```text
This archive contains derived source data, journal-facing supplementary tables and reproducibility support files for a public multi-omic glioma study centered on LAP3 as a benchmark-supported anchor of a malignant-microenvironment state. The package supports Figures 1-5 and Supplementary Figure S4A, including panel-level source data, key-results tables, panel maps, export-QC tables, frozen LAP3-state gene sets, submodule assignments, Core GBmap donor-state summaries, GBM-Space topology summaries, CPTAC boundary analyses and GLASS paired longitudinal summaries.

The archive contains only derived tables and manuscript-supporting outputs generated by the study. It does not redistribute controlled-access raw data or third-party raw omics files. GBM-Space raw Visium data should be accessed through EGA dataset EGAD00001015527 and study EGAS00001005801, with Data Access Committee EGAC00001000205. GLASS source data should be accessed through the original Synapse resources. TCGA, CGGA, Core GBmap and CPTAC source data should be accessed through their original repositories and publications.
```
